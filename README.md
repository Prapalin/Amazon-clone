# Amazon Clone (Frontend)

This is a frontend clone of the Amazon.com homepage built using HTML and CSS for practice purposes. 
It replicates the basic UI layout and design of Amazon.

## Features
- Amazon-like homepage UI
- Navigation bar with search box
- Product sections
- Footer design

## Tech Stack
- HTML
- CSS

## Project Structure
amazon-clone/
├── index.html
├── style.css
├── components/

## Note
This project is built for learning and practice. It is optimized for desktop view only.
